# 播放页操作（桌面端）

本页说明桌面端播放页的常用操作与快捷键（以实际版本为准）。快捷键可在 `设置 → 交互设置 → 桌面键盘/鼠标` 自定义（支持 Ctrl / Alt / Shift / Meta 组合）。

## 底部状态栏：音量 / 亮度

底部状态栏左下角提供两个条形滑块：

- 音量：0%–100%（播放器音量）
- 亮度：20%–100%（仅播放画面“视觉亮度”遮罩，不影响系统亮度）

两者都支持鼠标拖动滑块进行线性调节。

## 键盘（桌面端）

以下为默认键位（可自定义）：

- `Space`：播放 / 暂停
- `← / →`：快退 / 快进
- `↑ / ↓`：调节当前选中的「音量/亮度」
  - 默认选中音量
  - 鼠标悬停或点击某个条后，该条会成为当前控制目标
  - 单位时间内触发次数越多，单次增减幅度越大；停一会后会恢复为较小步长
- `Shift + ↑ / ↓`：直接调节亮度（不依赖当前选中条）

## 服务器播放额外快捷键（桌面端）

在服务器播放页（Emby / Jellyfin / WebDAV / Plex 等）额外支持：

- `F`：全屏 / 退出全屏
- `R`：线路切换
- `V`：版本选择（仅在线播放）
- `A`：音轨选择
- `S`：字幕选择
- `D`：弹幕面板
- `E`：选集面板

## 鼠标侧键（桌面端）

支持常见鼠标侧键（后退键 / 前进键）。默认行为：

- 后退键：快退
- 前进键：快进

可在 `设置 → 交互设置 → 桌面键盘/鼠标` 中改为「播放/暂停」或「不处理」。

## 快捷键自定义

在 `设置 → 交互设置 → 桌面键盘/鼠标` 中：

- 点击某个动作后，直接按下新的按键组合即可保存
- 按 `Backspace` / `Delete`：清除该动作的键位
- 按 `Esc`：取消并保留原设置

注意：尽量避免多个动作绑定到同一个按键组合；冲突时可能只有其中一个会生效。
